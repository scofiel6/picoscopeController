Data Conversion Functions

  • adc2mv: Converts raw Analog to Digital Converter count values to millivolts.
  • freq2delta: Convert frequency to delta value for PicoScope Arbitrary Waveform Generators.
  • mv2adc: Converts millivolt values to Analog to Digital Converter count values.
  • normalise: Normalize a vector or matrix.

Other Functions

  • getVerticalAxisLabel: Obtain a label to use on a vertical axis of a plot
  • importAWGFile: Import numeric data from a text file as column vectors.
  • questionDialog: Dialog prompting a User to respond 'Yes' or 'No' to a question.
  • stopButton: Displays a stop button.
  • timeunits: Returns a string representation of the time unit index.
